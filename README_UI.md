# Zapret UI

Zapret UI — это графический интерфейс для Windows, который управляет сборкой `zapret`.

Приложение рассчитано на простой сценарий: выбрать конфиг, запустить его, проверить состояние, выполнить диагностику, управлять списками и обновлять zapret без ручной работы в PowerShell.

## Основные функции

- Минималистичный интерфейс на русском языке.
- Запуск и остановка конфига одной кнопкой.
- Выбор стратегии внутри приложения.
- Диагностика стратегий с рекомендациями.
- Сохранение рекомендованных стратегий после перезапуска.
- Управление Game Filter из настроек.
- Обслуживание IPSet и hosts.
- Проверка и установка обновлений самого приложения Zapret.
- Автоматическая проверка обновлений этого форка.
- Скачивание, резервное копирование, установка и откат обновлений zapret.
- Переход в Telegram-бота для обратной связи.
- Иконка в трее.
- Кастомные окна подтверждения.
- Светлая и темная тема.

## Для пользователей

1. Скачайте архив последнего релиза:

```text
https://github.com/NIKOLAEV-ANDREI/zapret-ui/releases/latest
```

2. Распакуйте архив в простой путь, например:

```text
C:\zapret
```

3. Запустите:

```text
zapret.exe
```

Приложению нужны права администратора, потому что zapret использует WinDivert, службы Windows, изменения в реестре и сетевые команды.

## Обновления

Приложение отдельно проверяет:

```text
обновления приложения Zapret;
обновления файлов zapret.
```

Релизы берутся из этого репозитория:

```text
https://github.com/NIKOLAEV-ANDREI/zapret-ui/releases/latest
```

Перед установкой новой версии zapret создается резервная копия:

```text
backups/
```

Перед установкой новой версии приложения текущий `zapret.exe` сохраняется в:

```text
app-backups/
```

Пользовательские списки и настройки интерфейса сохраняются при обновлении.

## Обратная связь

Кнопка обратной связи находится в настройках приложения.

Ссылка на Telegram-бота хранится в файле:

```text
utils\feedback_bot.url
```

Пример:

```text
https://t.me/Zapret_UI_Feedback_bot
```

## Сборка интерфейса

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\ui\build.ps1 -OutDir ..\ui-build
```

Готовый файл:

```text
ui-build\zapret.exe
```
