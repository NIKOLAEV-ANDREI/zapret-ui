# Zapret

Zapret — это Windows-приложение с графическим интерфейсом для управления сборкой `zapret`.

Цель этого форка — сделать работу с zapret проще для обычного пользователя Windows: без ручного запуска PowerShell, без редактирования bat-файлов для стандартных действий и без необходимости запоминать, какой конфиг показал лучший результат.

## Возможности

- Запуск и остановка выбранного конфига одной кнопкой.
- Отображение текущего состояния: служба zapret, процесс `winws.exe`, IPSet и Game Filter.
- Выбор стратегий прямо внутри приложения.
- Встроенная диагностика стратегий.
- Отображение рекомендованных стратегий карточками на главном экране.
- Сохранение результатов диагностики после перезапуска приложения.
- Управление пользовательскими списками.
- Обновление IPSet и hosts.
- Проверка и установка обновлений самого приложения Zapret.
- Проверка обновлений этого форка.
- Скачивание и установка обновлений с созданием резервной копии.
- Откат на предыдущую версию zapret.
- Переход в Telegram-бота для обратной связи и сообщений об ошибках.
- Иконка в трее.
- Кастомные окна подтверждения.
- Светлая и темная тема.

## Скачать

Актуальная версия доступна на странице релизов:

```text
https://github.com/NIKOLAEV-ANDREI/zapret-ui/releases/latest
```

Скачайте `.zip`-архив, распакуйте его и запустите:

```text
zapret.exe
```

Приложение нужно запускать от имени администратора. Это необходимо, потому что zapret использует WinDivert, службы Windows, операции с реестром и сетевые команды.

## Обновления приложения, zapret и откат

В приложении есть два независимых типа обновлений:

```text
Обновление приложения — обновляет интерфейс Zapret.
Обновление zapret — обновляет файлы zapret, стратегии, списки и служебные компоненты.
```

Приложение проверяет обновления в этом репозитории:

```text
https://github.com/NIKOLAEV-ANDREI/zapret-ui/releases/latest
```

Перед установкой новой версии zapret приложение создает резервную копию в папке:

```text
backups/
```

Перед установкой новой версии приложения текущий `zapret.exe` сохраняется в папке:

```text
app-backups/
```

Если обновление zapret прошло неудачно или новая версия работает хуже, можно выполнить откат в настройках приложения.

Пользовательские списки и настройки интерфейса сохраняются при обновлении.

## Обратная связь через Telegram

В приложении есть кнопка для перехода в Telegram-бота обратной связи:

```text
Настройки -> Обратная связь -> Сообщить об ошибке
```

Ссылка на бота хранится в файле:

```text
utils\feedback_bot.url
```

Пример содержимого:

```text
https://t.me/Zapret_UI_Feedback_bot
```

## Сборка интерфейса

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\ui\build.ps1 -OutDir ..\ui-build
```

Готовый исполняемый файл будет создан здесь:

```text
ui-build\zapret.exe
```
